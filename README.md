### 🛠 Git Commit Message
|Type|Subject|
|---|---|
|**[FEAT]**|새로운 기능 구현|
|**[ADD]**|새로운 뷰, 부수적인 코드 추가, 에셋, 파일, 데이터 등 추가|
|**[FIX]**|버그 수정|
|**[CHORE]**|내부 파일 수정, 세팅 파일 수정|
|**[DEL]**|코드 삭제|
|**[DOCS]**|리드미 등 문서 수정|
|**[REFACTOR]**|코드 리팩토링, 전면 수정이 있을 때| 
|**[MERGE]**|머지|
|**[MOVE/RENAME]**|프로젝트 내 파일, 코드 이동 및 이름 변경|
